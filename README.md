# RemoteCtrl — Demo Control Panel

A React demo remote-control panel with animated activity flows.  
Every action simulates a connection sequence that always ends in a relay-server error.

## Stack
- React 18 + Vite
- Pure CSS (no UI library)

## Local dev
```bash
npm install
npm run dev
```

## Deploy to Vercel
Push this folder to a GitHub repo, then import it on [vercel.com](https://vercel.com).  
Vercel auto-detects Vite — no extra configuration needed.
